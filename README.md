# algorithm-study

## 계획

프로그래머스 뿌수기


### 방법

1. 풀 문제들에 관한 이슈 생성

   ```
   [sohyun] 1주차: 백준 수학 문제 풀기
   [ggyul] 1주차: Array 7문제 풀기
   [all] 2주차: 2021 어디 인턴 코테 풀기
   ```

2. 커밋하면서 메세지에 관련 이슈를 적는다. (#이슈번호) 

   ```
   solve: 수학 - 백준 5347 (#1)
   
   최대 공약수, 유클리드 호제법 // 추가 분류를 적을 수 있다면 적어준다.
   ```

3. 만나서 같이 한다 ^^

---
